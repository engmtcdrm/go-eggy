# Examples

This folder contains interactive examples demonstrating how to use the `go-eggy` package.

## Running the Examples

From this directory, simply run:

```bash
go run .
```

This will launch an interactive menu where you can select and run different examples showcasing eggy's features.

## What's Included

- **No Repeat Example** - Demonstrates a basic example prompt that exits after selecting one example
- **Continue to Prompt Example** - Shows how to use the `Repeat(true)` option to keep the prompt running after each example

## Structure

- `main.go` - Entry point that sets up and displays the example menu
- `examples/` - Contains all the example implementations
